# Investor Support System

A Java-based web platform designed to help investors monitor stock market data, manage portfolios, and visualize financial trends.

---

## Features

- User authentication system
- Portfolio management
- Watchlist tracking
- Stock price visualization
- Financial news integration
- Alert notification system
- Personal investment notes
- Administrative management panel

---

## System Architecture

The system follows the MVC architecture:

- Model: Data entities and DAO classes
- View: JSP, HTML, CSS, JavaScript
- Controller: Java Servlets

External APIs used:

- Finnhub API (stock price data)
- NewsAPI (financial news)

---

## Technologies Used

- Java Servlet & JSP
- MySQL Database
- Bootstrap 5
- TradingView Chart Widget
- Finnhub API
- NewsAPI
